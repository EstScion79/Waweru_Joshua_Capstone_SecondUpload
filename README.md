# Waweru_Joshua_Capstone_SecondUpload
Project Overview
Security Simplicity is an educational website dedicated to making cybersecurity concepts accessible and understandable for everyone. The project aims to demystify complex security topics and provide practical knowledge that users can apply immediately to enhance their online safety.

Website Purpose
Educate users about basic cybersecurity principles

Provide actionable security tips and best practices

Offer cybersecurity-related products and resources

Create a platform for security awareness and education
